====================================================================
 Snakemake workflows for working with Finnish NLS LiDAR survey data
====================================================================

Software requirements
=====================

- Snakemake (>= 3.5)
- libLAS with laszip support
- GDAL/OGR
- geotiff-bin
- ImageMagick
